# WolkAbout Angular Starter Project

Setup
-

1. Install NVM (Node Version Manager)
2. Use NVM to install the latest stable LTS Node version (This project was tested with 22.13.0)
   - Run `nvm list available`
   - Run `nvm install 22.13.0`
   - Run `nvm use 22.13.0`
3. Install pnpm: `npm install -g pnpm`
4. Run `pnpm install` to install all dependencies
5. Copy `src/config/config.example.yaml` into `src/config/config.yaml` and adjust as necessary.
6. Update `src/app/rest/authentication.api.ts` functions with appropriate URLs.
7. Run `pnpm start` to start the application

Development
-
Import `code-style.xml` and adhere to the coding standards ;)
